The Mob Book Idea

The basic premise of the mob book is anyone can create a story, it allows anyone to jump in and start writing the bits they want to write. As anyone can add a line at any point in the story and begin creating/collaborating their own there is never just one story to be told, the reader becomes an active part of the story choosing exactly what happens next without ever knowing ahead of time what will happen. 

Why I Started this

as I mentioned before I have been making desktop applications for a while, this is the first time I have really had a look into creating a website which actually does anything interesting, when I was looking for an excuse to learn how to build all this I had found an old chose your own adventure book. I remember thinking to myself "it’s a shame you can’t really chose, due to the limited options at any one time".

So now I had found my idea, all I had to do now was build it. It took a long time to find an interface between a programing language and html which I liked. I had started looking into asp. net as it was in the language I knew most c# but I had found this to clunky and difficult to get started in. so I continued looking, eventually I found the Flask library for python which where both powerful enough and simple enough for me to create something meaningful quickly, and with no prior knowledge of the library.

It’s been a fun journey finding the answers to all the little problems that have come up, and I am grateful to the people who I have bothered on the way for putting up with my questioning.
